# 🌱 AI-Платформа для личностного роста

> Видео-курсы + Тесты + ИИ-трекер привычек с персонализированными рекомендациями

## 📋 О проекте

**Проблема:** Нет единой российской платформы, объединяющей онлайн-обучение (видео-курсы), тестирование знаний и интеллектуальный трекер привычек с персонализированными ИИ-рекомендациями.

**Решение:** MVP платформы для личностного роста с сохранением прогресса, автоматической проверкой тестов и ИИ-советами.

## 🎯 Цели проекта

- ✅ 5 курсов с видеолекциями
- ✅ 3 теста для проверки знаний
- ✅ ИИ-трекер привычек с анализом прогресса
- ✅ Сохранение прогресса просмотра
- ✅ Автоматическая рекомендация следующего курса

## 🛠 Технологический стек

| Компонент | Технология |
|-----------|------------|
| Backend | Python 3.11 / FastAPI |
| Frontend | React 18 / TypeScript |
| Database | PostgreSQL 15 |
| Video Storage | Yandex Object Storage (S3) |
| AI | YandexGPT API |
| Containerization | Docker / Docker Compose |
| Version Control | Git / GitHub |
| CI/CD | GitHub Actions |
